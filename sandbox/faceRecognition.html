Face Recognition Sandbox - Readme

1. Create a Face Congitive resource in Azure

 The cloud-based Face API provides developers with access to advanced face algorithms. Microsoft Face algorithms enable face attribute detection and face recognition.

2. Obtain a Subscription Key

 This subscription key is used to access your Cognitive Services API. Do not share your key. Store your key securely - for example using Azure Key Vault. We also recommend regenerating your keys regularly, which can be done on the keys panel..
